Part 1:Identifying Usability Problems

Picking a Webpage


Original Bagel Gourmet Cafe Website Link

I chose this website because I like the breakfast sandwiches from Bagel Gourmet, but when I searched for the website, I realized that it is not designed as good as it can be.

Finding Problems

  • Key information such as the menu, location, and contact details are accessible, but it could be more prominently displayed
  • When used in a smaller screen, the screen does not adjust to the screen size
  • The lay out of the website is a bit awkward, there could be more of a flow to the content
  • The contrast between the bottom information and the background is not good and it might be difficult for users to see
  • There is a lot going on in the home page, so it is hard to know where to look

Accessibility

I agree with what WAVE detected. The contrast is not good, and could really use some improvement. Also, I noticed that WAVE detected that there are no alternative texts for the images. As we know, the alternative text is very important, as it describes the image for people who can not view the image and is a good alternative if the image is not loading.
